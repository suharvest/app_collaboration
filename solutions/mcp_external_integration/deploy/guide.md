# Deployment Guide

Use SenseCAP Watcher and MCP bridge to enable voice commands that directly call your business system APIs.

## Prerequisites

| Type | Requirement |
|------|-------------|
| Hardware | SenseCAP Watcher, Computer with Docker |
| Software | Docker Desktop installed and running |
| Account | [SenseCraft AI Platform](https://sensecraft.seeed.cc/ai/home) account |

## Deployment Process

Complete the 4 steps in order. Click the **Deploy** button for each step to automatically configure the component.

> Need help? Visit [Wiki Documentation](https://wiki.seeedstudio.com/cn/mcp_external_system_integration/).
