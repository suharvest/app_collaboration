Use SenseCAP Watcher and MCP bridge to enable voice commands that directly call your business system APIs.

## Prerequisites

| Type | Requirement |
|------|-------------|
| Hardware | SenseCAP Watcher, Device with Docker |
| Software | Docker installed and running |
| Account | [SenseCraft AI Platform](https://sensecraft.seeed.cc/ai/home) account |
