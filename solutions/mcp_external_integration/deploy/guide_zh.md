# 部署指南

使用 SenseCAP Watcher 和 MCP 桥接器，实现语音指令直接调用现有业务系统 API。

## 前提条件

| 类型 | 要求 |
|------|------|
| 硬件 | SenseCAP Watcher、运行 Docker 的电脑 |
| 软件 | Docker Desktop 已安装并运行 |
| 账户 | [SenseCraft AI 平台](https://sensecraft.seeed.cc/ai/home) 账户 |

## 部署流程

按顺序完成以下 4 个步骤，每个步骤点击**部署**按钮即可自动完成配置。

> 遇到问题？访问 [Wiki 文档](https://wiki.seeedstudio.com/cn/mcp_external_system_integration/) 获取帮助。
