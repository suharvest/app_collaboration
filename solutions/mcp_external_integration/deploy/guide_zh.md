## 部署流程预览

根据您选择的方案，按以下顺序完成部署：

### 私有云方案
```
1. 部署仓管系统 → 获取系统接口密钥
2. 部署语音服务 → 填写云端 AI 配置
3. 配置 SenseCraft → 获取设备连接地址
4. 启用连接桥接 → 打通语音和仓管系统
5. 测试体验 → 说话试试效果
```

### 边缘计算方案
```
1. 部署仓管系统 → 获取系统接口密钥
2. 部署 Jetson AI → 提供本地 AI 能力
3. 部署语音服务 → 连接 Jetson AI
4. 配置 SenseCraft → 获取设备连接地址
5. 启用连接桥接 → 打通语音和仓管系统
6. 测试体验 → 说话试试效果
```

## 部署前准备

| 类型 | 要求 |
|------|------|
| 硬件 | SenseCAP Watcher、能运行一键部署包的电脑 |
| 硬件（边缘方案） | reComputer J4012 或其他 Jetson 设备 |
| 软件 | 一键部署包已安装并运行 |
| 账户 | [SenseCraft AI 平台](https://sensecraft.seeed.cc/ai/home) 账户 |

## 方案对比

| 方案 | AI 服务 | 网络要求 | 响应速度 |
|------|---------|----------|----------|
| 私有云 | 您选的云服务 | 需联网 | ~0.8-1.2秒 |
| 边缘计算 | 本地 Jetson | 可离线 | ~1.0-1.5秒 |

## 常见问题

| 问题 | 解决方案 |
|------|---------|
| 一键部署包启动失败 | 检查是否有其他程序占用端口 |
| Watcher 连不上 WiFi | 确认是 2.4GHz 网络，不是 5GHz |
| 语音没反应 | 检查各服务状态是否正常 |
| Jetson 连接失败 | 检查 IP 地址和网络是否通畅 |

> 遇到其他问题？访问 [Wiki 文档](https://wiki.seeedstudio.com/cn/mcp_external_system_integration/) 获取帮助。
