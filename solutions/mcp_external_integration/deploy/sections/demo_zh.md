# 效果体验

恭喜！所有组件已部署完成，现在可以体验语音控制仓库管理系统了。

## 语音命令示例

| 您说 | 系统做什么 |
|------|-----------|
| "查询小智标准版的库存" | 查询并语音播报库存数量 |
| "入库 5 台 Watcher" | 增加库存，播报操作结果 |
| "出库 3 台发给销售" | 减少库存，生成出库记录 |
| "今天的库存汇总" | 汇总今日所有出入库数据 |

## 响应示例

**查询**："库存查询成功。Watcher 小智标准版当前有 150 台库存。"

**入库**："成功入库 5 台。之前 150 台，现在 155 台。"

## 典型使用场景

- 戴着手套操作时
- 驾驶叉车搬运时
- 高空作业时
- 双手被占用时

## 如何验证操作结果

打开 `http://localhost:2125` 进入仓库管理网页，查看库存数据是否已更新。

## 故障排除

| 问题 | 解决方案 |
|------|---------|
| 语音没反应 | 确认 Watcher 已配网并绑定平台 |
| 听不懂指令 | 说慢一点，语句清晰完整 |
| 操作未生效 | 检查连接桥接服务是否正常运行 |
| 网页数据没更新 | 刷新页面，或检查仓库系统服务状态 |
