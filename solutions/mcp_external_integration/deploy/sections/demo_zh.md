# 效果体验

所有组件已部署完成，现在可以体验语音控制仓库管理系统！

## 语音命令示例

| 类型 | 说法示例 |
|------|---------|
| 查询库存 | "查询小智标准版的库存" |
| 入库 | "入库 5 台 Watcher 小智标准版" |
| 出库 | "出库 3 台小智用于销售" |
| 统计 | "今天的库存汇总是什么？" |

## 响应示例

**查询**："库存查询成功。Watcher 小智标准版当前有 150 台库存。"

**入库**："成功添加 5 台。之前数量 150，新数量 155。"

## 应用场景

- 戴手套操作时
- 驾驶叉车时
- 高空作业时

## 验证

打开 `http://localhost:2125` 查看仪表盘确认操作已生效。
