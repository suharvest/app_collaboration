# Local Deployment

Deploy the warehouse management system on your local computer.

## Requirements

- Docker Desktop installed and running
- At least 2GB free disk space
- Ports 2124 and 2125 available

## After Deployment

1. Open `http://localhost:2125`
2. Register the first admin account
3. Create API key in **User Management** -> **API Key Management**
4. Copy the API key (format: `wh_xxxx...`) for MCP configuration
