# 语音服务本机部署

在本机部署语音服务，用于处理仓库管理语音指令。

## 服务说明

| 服务 | 功能 |
|------|------|
| 语音识别 | 将您说的话转成文字 |
| AI 对话 | 理解您的意图，调用对应功能 |
| 语音播报 | 把结果念出来告诉您 |

## 方案说明

### 私有云方案
- 语音识别：本地处理
- AI 对话：使用您填写的云服务（DeepSeek、OpenAI 等）
- 语音播报：使用免费在线服务

### 边缘计算方案
- 语音识别：本地处理
- AI 对话：连接本地 Jetson 设备
- 语音播报：连接本地 Jetson 设备

## 部署前检查

| 检查项 | 要求 |
|--------|------|
| 一键部署包 | 已安装并运行 |
| 磁盘空间 | 至少 3GB 可用 |
| 私有云方案 | 需要互联网连接 |
| 边缘计算方案 | 需要先部署好 Jetson 设备 |

## 响应速度参考

| 方案 | 说话到回复 |
|------|------------|
| 私有云 | 约 0.8-1.2 秒 |
| 边缘计算 | 约 1.0-1.5 秒 |

## 故障排除

| 问题 | 解决方案 |
|------|---------|
| 服务启动失败 | 检查端口是否被占用 |
| 语音识别不准 | 确保环境安静，说话清晰 |
| AI 无响应 | 检查网络连接（私有云）或 Jetson 状态（边缘） |
