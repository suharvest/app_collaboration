## 私有云方案部署

自托管部署方案，适合对数据隐私和完全控制有要求的组织。

### 架构说明

| 组件 | 位置 | 说明 |
|------|------|------|
| SenseCAP Watcher | 现场 | 语音输入设备 |
| MCP 桥接器 | 本地网络 | 自托管 MCP 端点 |
| 仓库管理系统 | 本地网络 | 业务逻辑与 API |

### 前提条件

- 部署设备已安装 Docker
- 所有设备之间的本地网络连接
- 部署完成后无需外部互联网

### 部署步骤

1. **部署仓库系统** - 在服务器上启动仓库 Docker 容器
2. **部署 MCP 桥接** - 在本地运行 MCP 桥接服务（无云端依赖）
3. **配置 Watcher** - 将 Watcher 指向本地 MCP 端点

### 与云方案的主要区别

- 所有数据保留在本地网络内
- 不需要 SenseCraft 云平台账户
- MCP 桥接器作为本地 Docker 服务运行
- 初始配置延迟稍高，但运行时延迟更低
