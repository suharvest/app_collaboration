# 语音 AI 服务远程部署

通过 SSH 将语音 AI 服务部署到 reComputer R1100（仓管盒子）。

## 部署架构

```
+-------------+     +-----------------------------------+
|   Watcher   |---->|      reComputer R1100             |
|  语音输入    |     |  +---------------------------+    |
+-------------+     |  |   xiaozhi-server          |    |
                    |  |   - VAD (Silero)          |    |
                    |  |   - ASR (SherpaASR)       |    |
                    |  |   - LLM (云端/Jetson)      |    |
                    |  |   - TTS (Edge/Kokoro)     |    |
                    |  +---------------------------+    |
                    |  +---------------------------+    |
                    |  |   mcp-endpoint            |    |
                    |  |   对接仓库管理 API          |    |
                    |  +---------------------------+    |
                    +-----------------------------------+
```

## 连接信息

- SSH 默认端口：22
- 默认用户名：root（OpenWrt）或您设置的用户名

## 注意事项

1. 确保 R1100 已连接到网络
2. 首次部署需要拉取 Docker 镜像（约 800MB）
3. 边缘计算方案需先部署 Jetson LLM/TTS 服务
