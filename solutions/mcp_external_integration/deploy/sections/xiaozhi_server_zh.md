# 语音服务部署

在 reComputer R1100（仓管盒子）上部署语音处理服务和系统对接服务。

## 服务说明

| 服务 | 功能 |
|------|------|
| 语音处理 | 识别您说的话，理解意图，语音回复 |
| 系统对接 | 连接仓库管理系统，执行实际操作 |

## 部署前检查

| 检查项 | 要求 |
|--------|------|
| 一键部署包 | 已安装并运行 |
| 磁盘空间 | 至少 3GB 可用 |
| 端口 | 18000、18003、18004 未被占用 |
| AI 服务 | 云方案需要互联网，边缘方案需要 J4012 已部署 |

## 故障排除

| 问题 | 解决方案 |
|------|---------|
| 服务启动失败 | 检查端口是否被占用 |
| 语音识别失败 | 确保环境安静，说话清晰 |
| AI 无响应 | 检查网络（云方案）或 J4012 状态（边缘方案） |
| 系统对接失败 | 确认仓库管理系统正在运行 |
