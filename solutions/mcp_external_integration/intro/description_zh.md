## 这个方案能帮你做什么

让语音助手（小智）直接控制你的业务系统。比如仓库管理员戴着手套或驾驶叉车时，只需说一句"入库 5 台 Watcher"，系统就能自动完成入库操作。

## 核心价值

| 好处 | 具体说明 |
|------|---------|
| 解放双手 | 戴手套、开叉车、高空作业时都能操作系统 |
| 实时更新 | 语音说完，数据立刻同步到系统 |
| 通用对接 | 有系统接口的业务系统都能接入（ERP、WMS、MES 等） |
| 灵活部署 | 支持云端、私有云、完全本地三种模式 |

## 适用场景

| 场景 | 怎么用 |
|------|--------|
| 仓库盘点 | "小智有多少库存" → 语音播报库存数量 |
| 货物入库 | "入库 10 台 Watcher" → 自动更新库存 |
| 出库发货 | "出库 3 台发给某某" → 生成出库单 |
| 库存查询 | "今天入了多少货" → 汇总今日数据 |

## 使用须知

### 硬件要求

| 设备 | 用途 | 必需 |
|------|------|------|
| SenseCAP Watcher | 语音输入设备 | 是 |
| 运行一键部署包的电脑 | 部署仓管系统和语音服务 | 是 |
| reComputer J4012 | 本地 AI 推理（边缘方案需要） | 否 |

### 网络要求

- Watcher 需连接 2.4GHz WiFi（不支持 5GHz）
- 云方案需要互联网连接
- 边缘方案可完全离线运行

### 部署方案对比

| 方案 | AI 对话能力 | 语音播报 | 适合场景 |
|------|-------------|----------|----------|
| SenseCraft 云方案 | 云端 | 云端 | 快速体验，无需额外设备 |
| 私有云方案 | 自选云服务 | 免费在线服务 | 数据留在本地，按需付费 |
| 边缘计算方案 | 本地 Jetson | 本地 Jetson | 完全离线，数据不出厂 |
