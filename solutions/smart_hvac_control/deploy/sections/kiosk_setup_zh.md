## Kiosk 模式配置

Kiosk 模式允许暖通控制界面在设备启动时自动以全屏方式运行。

### 什么是 Kiosk 模式？

- 系统启动时自动启动 Web 应用
- 以全屏方式显示界面，隐藏浏览器控件
- 禁用屏幕保护和电源管理
- 适用于专用控制站或监控显示屏

### 配置方式

部署完成后，您可以在**设备管理**页面启用或禁用 Kiosk 模式：

1. 导航到设备管理页面
2. 找到已部署的应用
3. 切换 Kiosk 开关
4. 选择运行 Kiosk 模式的系统用户账号
5. 配置将自动应用

### 前提条件

- 设备上已安装桌面环境
- 已安装 Chromium 或 Firefox 浏览器
- 有用于运行 Kiosk 模式的系统用户账号

### 手动配置

如果您希望通过 SSH 手动配置：

**启用 Kiosk 模式：**
```bash
./configure_kiosk.sh <用户名> http://localhost:8280
```

**禁用 Kiosk 模式：**
```bash
./unconfigure_kiosk.sh <用户名>
```

配置完成后，重启设备以应用更改。
