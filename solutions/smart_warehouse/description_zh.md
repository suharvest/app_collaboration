## 这个方案能帮你做什么

仓库管理系统功能强大，但上手成本高——要培训、要记菜单在哪、要学会各种操作流程。很多仓管员宁愿先记在纸上，再找人帮忙补录系统。

这个方案把复杂的系统操作变成**说话**——说一句"入库 10 台 Watcher"就完成入库，说"A3 货架还有多少货"立刻得到答案。不用学习，开口就会用。

## 核心价值

| 好处 | 具体说明 |
|------|---------|
| 零学习成本 | 不用培训、不用记菜单，开口说话就能操作系统 |
| 数据实时准确 | 直接查询数据库，库存数据实时更新，不存在同步延迟 |
| 数据安全可控 | 支持纯本地局域网部署，数据不出厂区，无需连接外网 |
| 接入现有系统 | 已有 ERP/WMS 系统？通过简单封装即可接入，无需更换 |

## 适用场景

| 场景 | 怎么用 |
|------|--------|
| 收货入库 | 说"入库 5 台 Watcher"，系统自动登记，货放下就完事 |
| 拣货出库 | 说"出库 3 台发给某某公司"，自动生成出库单 |
| 盘点核对 | 说"今天入了多少货"，语音汇总当日数据 |
| 叉车作业 | 司机说"A3 货架还有多少货"，语音播报库存数量，不用下车 |

## 使用须知

### 核心硬件设备

| 设备 | 说明 | 必需 |
|------|------|------|
| SenseCAP Watcher | AI 语音助手，佩戴在身上说话和听回复 | ✓ 必选 |
| reComputer R1100 | 边缘网关，内置智慧仓管系统 | ✓ 必选 |
| NVIDIA AGX Orin | 边缘 AI 计算机，提供本地 LLM/TTS 能力 | 边缘方案需要 |
| reComputer R2000 | R1100 升级版，人脸识别场景选配（20人以上） | 可选 |

### 三种部署方案对比

| 方案 | 网络要求 | 设备组合 | 适合谁 |
|------|----------|----------|--------|
| **云方案** ⭐推荐 | 需要外网 | Watcher + R1100 | 想快速体验，低成本起步 |
| **私有云方案** | 需要外网 | Watcher + R1100 | 数据隐私要求高，调用自选云 API |
| **边缘计算方案** | 纯局域网 | Watcher + R1100 + AGX Orin | 完全断网，数据不出厂区 |

### 可选功能：人脸识别操作员身份验证

| 场景规模 | 推荐方案 |
|----------|----------|
| 轻量场景（≤10人） | Watcher 内置人脸识别 |
| 严肃场景（20人+） | 升级到 R2000（R1100 升级版，不是替换 Watcher） |

### 网络要求

- Watcher 需连接 2.4GHz WiFi（不支持 5GHz）
- 云方案和私有云方案需要互联网
- 边缘计算方案可完全断网运行
