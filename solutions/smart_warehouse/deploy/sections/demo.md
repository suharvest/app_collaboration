# Demo & Testing

All components are deployed. Now experience voice-controlled warehouse management!

## Voice Command Examples

| Type | Example |
|------|---------|
| Query | "Check stock for Xiaozhi standard" |
| Stock-In | "Stock in 5 Watcher Xiaozhi standard units" |
| Stock-Out | "Stock out 3 Xiaozhi for sales" |
| Statistics | "What's today's inventory summary?" |

## Response Examples

**Query**: "Stock query successful. Currently has 150 units in stock."

**Stock-In**: "Successfully added 5 units. Previous 150, new 155."

- Wearing gloves
- Driving forklift
- Working at heights
