# 智控台配置

语音服务部署完成后，需要登录智控台配置模型 API 和获取 MCP 端点地址。

## 第一步：登录智控台

1. 浏览器访问 `http://服务器IP:8002`
2. 首次访问需注册账号，**第一个注册的账号即为管理员**
3. 使用注册的账号登录

## 第二步：配置模型 API

登录后，进入「模型配置」菜单：

| 配置项 | 说明 | 推荐选择 |
|--------|------|---------|
| 大语言模型 | 选择并填入 API 密钥 | 智谱 AI（免费额度）或阿里百炼 |
| 语音合成 | 选择 TTS 服务 | 火山引擎流式 TTS 或灵犀 TTS |
| 语音识别 | 默认使用本地 FunASR | 无需配置 |

> 首次使用建议选择「智谱 AI」，有免费额度可体验

## 第三步：获取 MCP 端点

1. 进入「参数管理」菜单
2. 找到 `mcp_endpoint` 参数
3. 复制 MCP WebSocket 地址（格式：`ws://IP:8004/mcp_endpoint/mcp/?token=xxx`）
4. 此地址将在下一步填入仓管系统

## 故障排除

| 问题 | 解决方案 |
|------|---------|
| 无法访问智控台 | 检查 8002 端口是否开放 |
| 模型调用失败 | 确认 API 密钥正确，账户有余额 |
| MCP 端点连接失败 | 检查防火墙是否放行 8004 端口 |
