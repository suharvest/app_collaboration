## 边缘计算方案部署

所有处理在本地边缘设备完成 - 适合隔离网络环境。

### 架构说明

| 组件 | 位置 | 说明 |
|------|------|------|
| SenseCAP Watcher | 现场 | 语音输入设备 |
| reComputer J4012 | 现场 | 本地运行所有服务 |
| 仓库管理系统 | 现场 | 边缘设备上的业务逻辑与 API |

### 前提条件

- reComputer J4012 或同等边缘设备
- 边缘设备已安装 Docker
- Watcher 连接所需的本地 WiFi 网络

### 部署步骤

1. **部署所有服务到边缘设备** - 将仓库系统 + MCP 桥接部署到 reComputer
2. **配置本地网络** - 为 Watcher 设置本地 WiFi
3. **配对 Watcher** - 将 Watcher 连接到边缘设备的 MCP 端点

### 优势

- 无需互联网连接即可工作
- 语音指令延迟最低
- 数据完全隔离
- 适合安全或偏远设施

### 注意事项

- 需要更强大的边缘硬件
- 更新需要手动应用
- 仅限本地网络覆盖范围
