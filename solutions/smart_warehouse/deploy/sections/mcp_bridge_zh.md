# 连接桥接服务

将语音助手的指令转换为仓库系统的操作，是打通语音和业务系统的关键一环。

## 工作原理

```
您说的话 → Watcher → 云平台 → 连接桥接 → 仓库系统
```

简单说：您对 Watcher 说"入库 5 台"，桥接服务就会调用仓库系统的入库功能，完成实际操作。

## 需要填写的信息

| 信息 | 说明 | 从哪里获取 |
|------|------|------------|
| 设备连接地址 | 以 `wss://` 开头的地址 | 上一步从 SenseCraft 平台复制的 |
| 系统接口密钥 | 以 `wh_` 开头的密钥 | 仓库系统 → 用户管理 → 接口密钥管理 |

## 如何获取系统接口密钥

1. 打开仓库管理系统网页 `http://localhost:2125`
2. 首次使用需要注册管理员账户
3. 进入 **用户管理** → **接口密钥管理**
4. 点击 **创建密钥**，复制生成的密钥

## 故障排除

| 问题 | 解决方案 |
|------|---------|
| 连接超时 | 检查设备连接地址是否正确复制 |
| 网络被阻止 | 用手机热点测试，排除办公网络防火墙问题 |
| 提示未授权 | 检查系统接口密钥是否正确 |
| 指令无响应 | 确认仓库系统服务正在运行 |
