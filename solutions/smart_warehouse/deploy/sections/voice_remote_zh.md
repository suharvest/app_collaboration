# 语音服务远程部署

通过网络将语音服务部署到 reComputer R1100（仓管盒子）。

## 部署架构

```
+-------------+     +-----------------------------------+
|   Watcher   |---->|      reComputer R1100             |
|  语音输入    |     |  +---------------------------+    |
+-------------+     |  |   语音处理服务              |    |
                    |  |   - 语音识别               |    |
                    |  |   - AI 对话                |    |
                    |  |   - 语音播报               |    |
                    |  +---------------------------+    |
                    |  +---------------------------+    |
                    |  |   系统对接服务              |    |
                    |  |   连接仓库管理系统          |    |
                    |  +---------------------------+    |
                    +-----------------------------------+
```

## 部署前检查

| 检查项 | 要求 |
|--------|------|
| R1100 | 已开机并连接网络 |
| 网络 | 电脑能 ping 通 R1100 |
| SSH | 知道 R1100 的用户名和密码 |

## 连接信息

- 默认端口：22
- 默认用户名：root（OpenWrt 系统）

## 故障排除

| 问题 | 解决方案 |
|------|---------|
| 连接超时 | 检查 IP 地址是否正确，网络是否通畅 |
| 认证失败 | 确认用户名和密码正确 |
| 部署慢 | 首次需下载约 800MB，请耐心等待 |
| 边缘方案不工作 | 确保已先部署好 Jetson 设备的 AI 服务 |
