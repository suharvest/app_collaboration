## 这个方案能帮你做什么

想知道仓库里的叉车在哪？想让员工进入会议室自动签到？这个方案让你在手机或电脑上就能看到人员和物品的实时位置。

## 核心价值

| 好处 | 具体说明 |
|------|---------|
| 找东西快 | 打开网页就能看到设备位置，不用满仓库找 |
| 安全保障 | 追踪器带 SOS 按钮，紧急情况一键报警 |
| 自动考勤 | 进入区域自动签到，无需手动打卡 |
| 部署简单 | 信标贴墙上就行，不用布线 |
| 成本可控 | 比 UWB 方案便宜很多，中小企业用得起 |

## 两种定位精度，按需选择

| 定位方式 | 适合场景 | 精度 | 你需要的设备 |
|---------|---------|------|-------------|
| 三角定位 | 需要知道具体位置 | 1-3 米 | 每个区域 3 个以上信标 |
| 区域定位 | 只需知道"在不在" | 区域级 | 每个区域 1 个信标 |

**简单选择**：
- 需要在地图上看到移动轨迹 -> 选三角定位（多放几个信标）
- 只需要知道"人在哪个房间" -> 选区域定位（每个房间一个信标就够）

## 适用场景

| 场景 | 怎么用 |
|------|--------|
| 仓储物流 | 追踪叉车和货物位置，找东西不用满仓库跑 |
| 校园安全 | 学生佩戴追踪器，紧急情况一键 SOS 报警 |
| 会议签到 | 进入会议室自动签到，无需扫码打卡 |
| 设备防盗 | 贵重设备离开指定区域自动报警 |
| 养老院 | 追踪老人位置，防止走失 |

## 使用须知

**硬件要求**：
- 追踪器：SenseCAP T1000（被追踪的人或物佩戴）
- 信标：BC01 蓝牙信标（固定安装在墙上，作为定位参考点）
- 网关：SenseCAP M2 LoRaWAN 网关（接收追踪器信号并上传数据）
- 电脑：安装 Docker，运行定位服务

**环境要求**：
- 信标安装高度 2.5-3 米
- 信标间距 10-15 米
- 网关覆盖半径约 2 公里

**精度说明**：
- 三角定位精度 1-3 米（取决于信标密度和环境）
- 金属遮挡、人员密集会降低精度
