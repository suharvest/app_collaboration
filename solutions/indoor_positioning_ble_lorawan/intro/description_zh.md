## 概述

室内定位是许多行业面临的常见挑战。虽然 UWB 等高精度系统存在，但它们成本较高且部署复杂。本解决方案通过结合两种强大的无线技术，提供灵活且经济高效的替代方案：**蓝牙低功耗（BLE）** 用于位置感知，**LoRaWAN** 用于远距离、低功耗数据传输。

该系统以 SenseCAP T1000 追踪器为核心，支持两种不同的定位模式：

| 模式 | 描述 | 最佳应用场景 |
|------|------|-------------|
| **三角定位** | 检测到 3+ 个信标时计算精确 (x, y) 坐标 | 资产/人员移动追踪 |
| **邻近定位** | 根据最近的信标识别位置 | 签到、在场检测 |

## 核心特性

- **经济高效且可扩展** - 使用经济实惠的 BLE 信标，无需在每个房间安装昂贵的网关设备
- **远距离传输** - 单个 LoRaWAN 网关可覆盖整栋建筑或园区
- **即时 SOS 报警** - 紧急按钮触发即时警报并显示最后已知位置
- **智能电源管理** - 运动激活上报模式可将电池寿命延长至 6 个月
- **简单部署** - 快速设置 BLE 信标和 LoRaWAN 网关
- **开源** - 完全可定制的后端，GitHub 仓库已公开

## 使用场景

### 校园安全
为学生和教职员工提供可穿戴追踪器。内置 SOS 按钮允许从校园任何地方发送即时紧急警报。

### 资产管理
追踪室内和室外的贵重资产。配置追踪器仅在检测到移动时上报，在节省电池的同时确保未经授权的移动能触发警报。

### 自动签到与考勤
通过调整 BLE 信标信号强度，为办公室或养护设施创建"签到"区域。当携带追踪器的人员进入该区域时，自动记录其在场状态。

## 系统架构

该系统基于简单而强大的原理运作：

1. **BLE 信标** - 放置在已知位置的固定发射器，持续广播唯一 ID
2. **SenseCAP T1000 追踪器** - 移动设备，扫描附近信标并识别最强信号
3. **LoRaWAN 网关** - 接收包含最近信标 ID 的追踪器数据包
4. **应用服务器** - 将信标 ID 映射到物理位置并可视化追踪器位置
