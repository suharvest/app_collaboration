## 这个方案能帮你做什么

仓库里的叉车去哪了？员工有没有按时到岗？贵重设备是不是被挪走了？传统方案要么布线麻烦、要么太贵。这套方案用蓝牙定位标签 + 远距离无线传输，让你在电脑或手机上随时查看人员和物品的位置。

## 核心价值

| 好处 | 具体说明 |
|------|---------|
| 找东西快 | 打开网页输入设备名，几秒钟就知道在哪个区域 |
| 免布线 | 定位标签贴墙上就行，电池能用 2 年 |
| 覆盖广 | 一个接收器覆盖 2 公里，整个园区都能管 |
| 自动记录 | 进出区域自动打卡，不用手动签到 |
| 一键求助 | 追踪器带 SOS 按钮，紧急情况长按报警 |

## 适用场景

| 场景 | 怎么用 |
|------|--------|
| 仓库找货 | 叉车、货物挂上追踪器，网页地图直接看位置，不用满场跑 |
| 工厂巡检 | 巡检员带追踪器，系统自动记录路线和时间，省去手工填表 |
| 校园安全 | 学生佩戴追踪器，遇到危险长按 SOS，老师立刻收到位置 |
| 养老院防走失 | 老人佩戴追踪器，离开安全区域手机自动提醒 |
| 设备防盗 | 贵重仪器贴上追踪器，被移动时自动报警 |

## 使用须知

**需要准备的硬件**：
- 追踪器（T1000）：挂在要追踪的人或物上，卡片大小，带挂绳
- 蓝牙定位标签（BC01）：贴在墙上作为参考点，越多定位越准
- 接收网关（M2）：放在中心位置，接收追踪器信号并上传数据
- 电脑一台：运行定位软件，可以是普通办公电脑

**两种定位精度**：
- 精确定位（1-3米）：每个区域放 3 个以上定位标签，可以在地图上看移动轨迹
- 区域定位：每个房间放 1 个定位标签，只显示"在哪个房间"

**安装建议**：
- 定位标签安装高度 2.5-3 米
- 标签间距 10-15 米
- 金属货架多的地方要多放几个标签
