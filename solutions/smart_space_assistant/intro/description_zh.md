## 这个方案能帮你做什么

你家的AI助手很聪明，但它"看不见"——不知道是谁在跟它说话。

想象一下：你下班回家，AI助手能认出你的脸，主动说"欢迎回来，小明！今天有3封重要邮件"。来客人了，它能分辨出是家人还是陌生人。在展厅，它能认出VIP客户，提供专属服务。

这个方案给你的AI助手装上"眼睛"和"大屏"，让它真正成为你的空间智能管家。

## 核心价值

| 好处 | 具体说明 |
|------|---------|
| 认人打招呼 | 说"记住我的脸，我叫小明"，下次进门自动问候 |
| 个性化服务 | 不同的人进来，说不同的话、做不同的事 |
| 隐私在手中 | 所有人脸数据存在本地设备，不传到云端 |
| 大屏展示 | 把对话内容投到电视/大屏，多人一起看 |

## 适用场景

| 场景 | 怎么用 |
|------|--------|
| 智能家庭 | 家人进门自动问候，孩子回家提醒作业，老人起床关心身体 |
| 会客接待 | 识别VIP客户，自动推送专属服务信息到大屏 |
| 展厅展示 | AI互动大屏吸引参观者，认脸后记住偏好下次继续聊 |
| 办公室 | 早上认出你打招呼，顺便告诉你今天的日程安排 |

## 使用须知

### 人脸识别功能

| 条件 | 说明 |
|------|------|
| 容量 | 最多记住20个人的脸 |
| 光线 | 需要正常室内光线，太暗会认不出 |
| 角度 | 正脸效果最好，侧脸可能认不准 |
| 设备 | 只需要SenseCAP Watcher，你已有的设备即可 |

### 大屏投屏功能

| 条件 | 说明 |
|------|------|
| 显示器 | 需要一台支持HDMI的电视或显示器 |
| 计算设备 | 需要一台电脑或树莓派来运行投屏服务 |
| 网络 | Watcher和投屏设备要连同一个WiFi |

### 两个功能可以分开装

- 只想要认人功能？只装人脸识别
- 只想要大屏？只装投屏功能
- 都想要？两个一起装，互不影响
