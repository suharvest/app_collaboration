## 方案概述

本方案在 reCamera 设备上部署实时人流检测和热力图可视化系统。通过 YOLO26 模型，可以检测和跟踪零售环境中的人员，分析其行为模式，并生成动态热力图显示高流量区域。

## 核心特点

- **实时检测** - YOLO26 模型直接在 reCamera 边缘设备上运行
- **行为分析** - 自动分类人员状态（浏览、停留、需要帮助）
- **热力图可视化** - 动态热力图展示人员停留最多的区域
- **隐私优先** - 所有处理在设备端完成，视频不离开门店
- **易于集成** - MQTT 输出，方便与其他系统集成

## 应用场景

| 场景 | 说明 |
|------|------|
| 店铺布局优化 | 识别高流量区域和冷门区域，优化商品摆放 |
| 客户服务 | 当顾客可能需要帮助时（长时间停留）提醒店员 |
| 转化分析 | 追踪浏览与深度参与的行为模式 |
| 排队管理 | 监控收银区域和等待时间 |

## 技术规格

| 组件 | 规格 |
|------|------|
| 检测模型 | YOLO26n (INT8 量化) |
| 推理速度 | 约 300ms/帧 |
| 输出格式 | MQTT JSON 消息 |
| 视频流 | RTSP (端口 8554) |
| 跟踪 | 多目标跟踪，带状态分析 |

## 人员状态

系统将每个检测到的人员分为四种状态：

- **移动中** - 正在穿过区域（速度 > 10 px/s）
- **短暂停留** - 短暂停止（< 1.5 秒）
- **深度参与** - 正在浏览/感兴趣（停留 1.5s - 20s）
- **需要帮助** - 可能需要帮助（停留 > 20 秒）
