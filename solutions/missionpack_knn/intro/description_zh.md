## 这个方案能帮你做什么

办公楼、商场、工厂的空调系统往往"傻傻地开着"——不管有没有人、外面冷不冷，一直按固定温度运行。结果就是：没人的区域也在吹冷气，大热天空调却不够凉。这个方案让空调系统"变聪明"，能根据人流多少和环境变化自动调节温度，该开就开、该省就省。

## 核心价值

| 好处 | 具体说明 |
|------|---------|
| 自动省电 | 根据实时人流和室外温度智能调节，通常可节省 15-25% 电费 |
| 无需值守 | 系统 24 小时自动运行，不用人盯着调参数 |
| 快速上手 | 导入历史数据后系统自动学习，不需要专业暖通知识 |
| 安全可控 | 支持"先看后做"模式，确认合理再开启自动控制 |

## 适用场景

| 场景 | 怎么用 |
|------|--------|
| 写字楼 | 工作日白天人多时自动调低温度，下班后自动升温节能；周末自动进入节能模式 |
| 商场 | 根据各楼层客流量分区调节，人少的区域减少制冷，人多的区域加强降温 |
| 工厂车间 | 结合生产排班自动调节，停产时段提前降低空调功率 |
| 酒店大堂 | 根据入住率和室外温度动态调整，既保证舒适又节省能耗 |

## 使用须知

**硬件要求**：
- 一台装有 Docker 的电脑或工控机（可用 reComputer R 系列）
- 空调控制器需支持工业设备通讯协议（OPC-UA）

**数据要求**：
- 需要至少 1 周的历史运行数据（Excel 或 CSV 格式）
- 数据包含：时间、温度设定值、实际温度、用电量

**使用限制**：
- 首次使用需上传数据让系统学习，约需 5-10 分钟
- 建议先观察 1-2 天系统建议，确认合理后再开启自动控制
- 本方案适用于中央空调系统，不适用于分体式空调
