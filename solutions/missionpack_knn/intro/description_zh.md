## 这个方案能帮你做什么

暖通空调系统（冷水机、空调机组等）的参数调节通常依赖老师傅的经验，新手操作容易造成能源浪费，老师傅也难以24小时盯守。这个方案能根据历史运行数据，自动学习最佳参数组合，让设备像有经验的老师傅一样自动调参。

## 核心价值

| 好处 | 具体说明 |
|------|---------|
| 省电省钱 | 通过分析历史数据找到最优运行参数，通常可降低 10-20% 能耗 |
| 减少人工 | 系统自动给出参数建议或直接执行调节，无需人工值守 |
| 新手也能用 | 不懂暖通的运维人员也能快速上手，系统会告诉你该怎么设 |
| 安全验证 | 支持"只看不动"模式，先观察建议值是否合理，确认后再自动执行 |

## 适用场景

| 场景 | 怎么用 |
|------|--------|
| 商场/写字楼 | 根据室外温度和室内负荷，自动调节冷冻水温度和风机频率 |
| 工厂车间 | 结合生产计划预测空调需求，提前调整避免温度波动 |
| 数据中心 | 精确控制机房温湿度，保障设备安全运行 |
| 酒店 | 根据入住率动态调整公共区域空调参数 |

## 使用须知

**硬件要求**：
- 装有 Docker 的电脑或工控机（如 reComputer R 系列）
- 暖通设备需支持 OPC-UA 协议读写参数

**数据要求**：
- 需要至少 1 周的历史运行数据（CSV 格式）
- 数据应包含：时间、设定参数、实际运行值、能耗数据

**限制说明**：
- 首次使用需要上传数据"训练"系统，过程约需 5-10 分钟
- 建议先用"只看不动"模式运行 1-2 天，确认建议合理后再开启自动控制
