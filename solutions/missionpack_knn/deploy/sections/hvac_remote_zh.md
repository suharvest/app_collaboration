## 远程部署

通过 SSH 将暖通自动化控制系统部署到远程设备。

### 目标设备要求

- 基于 Linux 的系统（Ubuntu、Debian 等）
- 已安装并运行 Docker
- 已启用 SSH 访问
- 端口 8280 和 4841 可用

### 连接信息

- **设备 IP**：远程设备的 IP 地址
- **SSH 用户名**：登录用户名（默认：recomputer）
- **SSH 密码**：登录密码

### 部署内容

部署过程将：
1. 通过 SSH 连接到远程设备
2. 在 `/home/<用户>/missionpack_knn` 创建部署目录
3. 上传 Docker Compose 文件
4. 拉取容器镜像
5. 启动应用服务

### 部署完成后

应用将在 **http://<设备IP>:8280** 可用

您还可以配置 Kiosk 模式，使应用在开机时自动全屏启动。
