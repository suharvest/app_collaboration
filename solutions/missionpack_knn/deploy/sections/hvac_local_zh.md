## 本机部署

使用 Docker 在本机部署暖通自动化控制系统。

### 部署内容

- **暖通控制应用** - Web 界面和 API 服务器（端口 8280）
- **内置 OPC-UA 模拟器** - 用于无实际设备时的测试（端口 4841）

### 配置说明

**OPC-UA 服务器地址**：输入您的 OPC-UA 服务器地址，或保持默认值使用内置模拟器。

### 部署完成后

应用将在 **http://localhost:8280** 可用

您的数据将持久化存储在 Docker 卷中：
- `configs/` - 配置文件
- `uploads/` - 上传的训练数据
- `models/` - 训练好的预测模型
- `log/` - 应用日志
