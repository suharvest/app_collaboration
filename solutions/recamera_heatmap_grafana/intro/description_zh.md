## 这个方案能帮你做什么

想知道店里哪个区域人最多？这个方案把 reCamera 拍到的人流数据变成直观的人流分布图，还能自动打码保护顾客隐私。

## 核心价值

| 好处 | 具体说明 |
|------|---------|
| 看懂人流走向 | 不用盯着监控，直接看图就知道哪里人多哪里人少 |
| 自动保护隐私 | 人像在摄像头端就被打码，传出来的数据不含人脸 |
| 数据保存在本地 | 所有数据存在你的电脑上，不上传云端 |
| 历史对比分析 | 可以看今天和昨天、本周和上周的人流变化 |

## 适用场景

| 场景 | 怎么用 |
|------|--------|
| 零售门店 | 看哪个货架前人最多，调整商品陈列和促销位置 |
| 展会活动 | 找出最热门的展位，为下次布展提供数据 |
| 办公空间 | 分析工位和会议室使用率，优化空间规划 |
| 餐厅咖啡馆 | 了解哪些座位受欢迎，调整桌椅布局 |

## 使用须知

**硬件要求**
- reCamera 摄像头一台
- 一台装有 Docker 的电脑（Windows/Mac/Linux 均可）

**网络要求**
- reCamera 和电脑需连接同一 WiFi

**隐私说明**
- 人像在 reCamera 本地就被打码，传输的只有坐标数据
- 所有数据保存在你的电脑上，不上传任何服务器
