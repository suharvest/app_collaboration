## 核心价值

- **边缘 AI 处理** - 所有人员检测都在 reCamera 本地使用 YOLO11n 模型运行，无需云端
- **隐私保护** - 在数据传输前自动对检测到的人员进行模糊处理
- **实时分析** - 实时热力图可视化，展示人流模式
- **企业级仪表板** - 使用 Grafana 和 InfluxDB 时序数据库实现专业数据可视化

## 技术亮点

| 特性 | 描述 |
|------|------|
| 检测模型 | 针对边缘计算优化的 YOLO11n |
| 数据流 | reCamera 上的 Node-RED 工作流引擎 |
| 时序数据库 | InfluxDB 2.x 高性能数据存储 |
| 可视化 | Grafana 仪表板实时更新 |
| 热力图 | 基于 HTML5 Canvas 的热力可视化 |

## 应用场景

| 场景 | 用途 |
|------|------|
| 零售分析 | 追踪客户流量模式和热点区域 |
| 办公空间 | 监测工位利用率并优化布局 |
| 展览馆 | 识别热门展品和访客流向 |
| 公共场所 | 人群密度监测和安全管理 |

## 系统架构

该解决方案由三个主要组件组成：

1. **reCamera** - 运行人员检测和隐私模糊的边缘 AI 设备
2. **InfluxDB** - 存储检测坐标的时序数据库
3. **Grafana** - 显示实时热力图和统计数据的仪表板

数据从 reCamera 通过 Node-RED 流向 InfluxDB，Grafana 查询数据库进行可视化。
