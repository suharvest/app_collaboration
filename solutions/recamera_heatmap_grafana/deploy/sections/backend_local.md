# Local Deployment

Deploy InfluxDB and Grafana on your local computer.

## Requirements

- Docker Desktop installed and running
- At least 2GB free disk space
- Ports 8086 and 3000 available

## After Deployment

Access InfluxDB: `http://localhost:8086`
- Credentials: admin / adminpassword
- Organization: seeed, Bucket: recamera

Go to **API Tokens** and copy the token for later configuration.

Access Grafana: `http://localhost:3000`
- Default credentials: admin / admin
