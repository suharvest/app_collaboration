### Configure reCamera

1. Visit [SenseCraft reCamera](https://sensecraft.seeed.cc/ai/recamera), deploy heatmap app to reCamera
2. In Node-RED, install `node-red-contrib-influxdb` node
3. Configure InfluxDB node: URL `http://<server-ip>:8086`, paste Token
4. Click Deploy, verify data in InfluxDB Data Explorer
