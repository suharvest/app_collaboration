## 部署概述

本方案需要两个部署步骤：

1. **烧录 Watcher 固件** - 为 Watcher 烧录支持投屏的固件
2. **部署投屏服务** - 在 reComputer/树莓派上运行投屏服务器

## 部署前准备

开始部署前，请确认：

- [ ] SenseCAP Watcher 和 USB-C 数据线
- [ ] 已安装 Docker 的 reComputer 或树莓派
- [ ] 连接到计算设备的 HDMI 显示器
- [ ] 所有设备在同一网络

## 部署完成后

两个步骤都完成后：

1. 在显示设备上打开浏览器，访问 `http://localhost:8765`
2. 按 `F` 或 `F11` 进入全屏模式
3. 配置 Watcher 连接到投屏服务器的 IP 地址
4. 开始对话 - 内容将显示在大屏幕上！
