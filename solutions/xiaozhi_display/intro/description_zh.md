## 概述

小智投屏将 SenseCAP Watcher AI 助手的内容投射到大屏幕，适用于：

- **户外广告** - 带 AI 互动功能的数字标牌
- **零售展示** - 带语音助手的产品展示屏
- **会议室** - 共享 AI 助手的会议室大屏
- **智能家居中枢** - 家庭自动化的中央显示屏

## 工作原理

系统采用 **UI 状态同步 + Web 渲染** 架构：

1. **Watcher 设备** 通过 WebSocket 发送 UI 状态（表情、文字、聊天消息）
2. **投屏服务器** 接收状态并广播给浏览器客户端
3. **Web 浏览器** 渲染带动画效果的 UI 界面
4. **音频输出** 解码（Opus → PCM）后通过 HDMI/扬声器播放

## 核心特性

| 特性 | 说明 |
|------|------|
| **多客户端支持** | 多个浏览器可同时查看 |
| **mDNS 自动发现** | Watcher 自动发现网络中的投屏服务器 |
| **可定制主题** | 支持自定义背景和表情素材 |
| **音频放大** | 通过 HDMI 将音频路由到外部扬声器 |
| **低延迟** | WebSocket 实时 UI 同步 |

```
                                                │ HTTP + WebSocket

                                       ┌──────────────────┐

                                                │ HDMI
                                                ▼
                                       ┌──────────────────┐
                                       │ 电视 / 显示器 /  │
                                       │    广告机        │
                                       └──────────────────┘
```

## 硬件要求

- SenseCAP Watcher（需烧录定制固件，已包含）
- reComputer R 系列 或 树莓派 4/5
- HDMI 显示器（电视、显示器或广告机）
- 网络连接（WiFi 或以太网）
