# 远程部署

将语音服务部署到远程设备（reRouter、树莓派等）。

## 开始之前

1. **将目标设备连接到网络**
   - 确保设备与您的电脑在同一网络
   - 记录设备的 IP 地址（OpenWrt 默认：192.168.49.1）

2. **获取设备凭据**
   - SSH 用户名（OpenWrt 通常是 `root`）
   - SSH 密码（OpenWrt 默认为空）

## 连接设置

填写以下信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| 设备 IP | 目标设备 IP 地址 | 192.168.49.1 |
| SSH 密码 | 登录密码（可选） | your-password |

## 部署完成后

1. SSH 执行 `docker ps` 确认三个容器运行中
2. 访问 `http://192.168.49.1:8090` 打开边缘客户端
3. 开始实时语音转录测试
