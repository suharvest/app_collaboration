## 这个方案能帮你做什么

在零售门店里监督服务质量，不用派人盯着，麦克风自动采集对话，AI 分析店员有没有按规范服务。

## 核心价值

| 好处 | 具体说明 |
|------|---------|
| 服务质量可量化 | 自动转录店员和顾客的对话，分析服务规范执行情况 |
| 隐私有保障 | 语音在本地设备上转成文字，只上传文本不传录音 |
| 多店集中管理 | 一个仪表板看所有门店，支持关键词热点分析 |
| 部署成本低 | 一台小盒子加一个麦克风，几百块搞定 |

## 适用场景

| 场景 | 怎么用 |
|------|--------|
| 连锁门店 | 监测店员服务话术是否规范 |
| 培训考核 | 新员工服务流程录音回放 |
| 客诉分析 | 自动标记投诉关键词，快速定位问题 |
| 销售分析 | 统计产品被问到的频率，辅助选品 |

## 使用须知

| 类型 | 要求 |
|------|------|
| 边缘设备 | reRouter CM4（4GB 内存，32GB 存储） |
| 麦克风 | reSpeaker XVF3800 四麦克风阵列 |
| 网络 | 有线网络，需连接互联网上传分析结果 |
| 环境 | 背景噪音不宜过大，建议 70dB 以下 |
